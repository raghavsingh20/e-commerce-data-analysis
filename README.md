# E-Commerce-Data-Analysis

# Overview

Analyzed 120K+ e-commerce order records using SQL to explore business performance, customer behavior, and operational trends. The project focuses on extracting insights from transactional data using real-world business analysis scenarios.

# Dataset

The dataset contains e-commerce transactional order data, including:
Order details, Product information, Customer type, Shipping details etc

Total records: 120K+
Dataset source: https://www.kaggle.com/datasets/mdsazzatsardar/amazonsalesreport
